README

This folder contains code and data sufficient to reproduce the results in the study “Improved prediction of radiation-induced hypothyroidism in nasopharyngeal carcinoma using pre-treatment CT radiomics” by Ritlumlert et al.

File
1) LR_combined_model.ipynb
Description: Code for Code for combined logistic regression.

2) RF_combined_model.ipynb
Description: Code for combined random forest.

3) input_LR.xlsx
Description: Input data for combined logistic regression.

4) input_RF.xlsx
Description: Input data for combined random forest.

5) outcome.xlsx
Description: Outcome data (1 = Radiation induced hypothyroidism, 
			 0 = no radiation induced hypothyroidism) 

6) Feature_description.xlsx
Description: Variable descriptions
